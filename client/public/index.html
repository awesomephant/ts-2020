    Work A

    A speculative font editor/design tool for modular letterforms.

    Letterforms are created by placing markers on a predefined grid. The system then replaces these markers with geometric shapes to make up the letterform, and exposes controls to transform the shapes and the underlying grid.

    This was done as part of and ongoing personal project experimenting with home-made design tools.

    See the (work in progress) live app here: https://awesomephant.github.io/grid-system/
